本研究專案專注於 Realsense 姿態偵測技術應用於步態分析。 This project focuses on Gait Analysis using Realsense Pose Estimation at NYCU.

  • 步態特徵提取 (Gait Feature Extraction)
  • 每圈橫向偏移分析 (Lateral Offset Analysis)
  • 頻譜分析 (FFT/PSD Spectrum Analysis)
  • 骨盆朝向追蹤 (Pelvis Heading Tracking θ(t))
